# 개발 및 운영에 필요한 안내

어떤 포트를 사용하는 지 start.sh 스크립트 참고 부탁드립니다.
기본 포트인 3000은 개발 서버에서 이미 ISG가 shipsong 으로 사용중이므로 4000 을 사용하도록 되어 있으며, 사용 버전으로 띄울 때는 39000 포트를 쓰도록 되어 있습니다.

auth 관련된 부분은 무시하시고 전체 기능을 고도화 하는 것에 집중해주시고 로그인 관련된 내용은 정리되는 대로 함께 보완해서 구현하고자 합니다.

## build

```
build.sh dev
build.sh prd

```

### run
```
start.sh dev
start.sh prd
```
